Getiing Started

Follow these commands to get started

go get -v github.com/ramitsurana/turbo
cd $GOPATH/src/github.com/ramitsurana/turbo
./turbo

#For aliasing use
alias turbo="sudo '${GOPATH}/src/github.com/ramitsurana/turbo/turbo'"

Command Description
clean Cleans up all your docker images from your system
clear Clears all your exited or stopped docker containers from your system
monitor Monitors your docker containers
backup Creates a backup copy of your Docker Stuff
